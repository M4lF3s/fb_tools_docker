## Usage

Run the Container with:
```sh
docker run --rm -it -v ${PWD}:/tmp m4lf3s/fb_tools
```

And then inside the Container e.g.:
```sh
fb_tools [Password]@fritz.box konfig export-decrypt [Filename]
```
